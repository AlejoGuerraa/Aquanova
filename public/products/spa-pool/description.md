# Pileta con Spa Integrado

## Descripción General
La combinación perfecta de ejercicio y relajación. Esta pileta integra una zona de spa con hidromasaje, permitiéndote disfrutar de natación refrescante y masajes terapéuticos en el mismo espacio.

## Características Destacadas
- **Doble zona**: Área de natación y spa separadas pero integradas
- **Hidromasaje terapéutico**: Jets ajustables para masaje personalizado
- **Calefacción dual**: Controla independientemente la temperatura de cada zona
- **Diseño continuo**: Transición fluida entre ambas áreas

## Especificaciones Técnicas
- Zona de natación: 9m x 4.5m x 1.5m
- Zona de spa: 3m x 3m x 1.2m
- Capacidad total: 62,000 litros
- Material: Hormigón con acabado antideslizante

## Sistemas Incluidos
- 12 jets de hidromasaje ajustables en zona de spa
- Sistema de burbujas de aire
- Calefacción independiente para cada zona
- Iluminación LED cromática
- Control de química automatizado
- Filtrado de alta eficiencia
- Asientos ergonómicos integrados en spa

## Personalización
Configuramos la cantidad y posición de los jets según tus preferencias. El sistema de domotización permite programar sesiones de spa con temperatura, jets y luces personalizadas.

# Pileta Infinity con Vista

## Descripción General
El diseño infinity es la expresión máxima de elegancia y arquitectura moderna. El agua parece fluir sin límites, creando una conexión visual perfecta con el paisaje circundante.

## Características Destacadas
- **Borde infinito**: El agua se desborda de forma controlada creando el efecto infinity
- **Arquitectura moderna**: Líneas limpias y diseño minimalista
- **Ingeniería de precisión**: Sistema de recirculación perfectamente calibrado
- **Efecto visual único**: Parece fundirse con el horizonte

## Especificaciones Técnicas
- Largo: 12 metros
- Ancho: 5 metros
- Profundidad: 1.6 metros
- Capacidad: 72,000 litros
- Material: Hormigón con revestimiento de porcelanato

## Sistemas Incluidos
- Sistema de recirculación de borde infinity
- Tanque de compensación oculto
- Bomba de alta capacidad
- Calefacción por intercambiador
- Iluminación perimetral LED
- Control automático de nivel de agua
- Sistema de filtrado de doble etapa

## Personalización
Diseñamos el sistema infinity específicamente para tu terreno y vista. El kit de domotización controla todos los parámetros para mantener el efecto perfecto en todo momento.
